# Customization of the Dialogue Box

## Introduction

You can customize the dialogue box by modifying the template. However, please note that after modifying the template, do not update the Konado plugin, or your modifications will be overwritten.

> [!NOTE]
> Please DO NOT update Konado plugin after modifying the dialogue box, or your modifications will be overwritten.

## Editing the Scene File

`res://addons/konado/scenes/konado_dialogue.tscn` is the dialogue interface scene, and you can customize the dialogue interface by modifying this file.

In general, please refrain from modifying scripts on nodes, and customise effects by adjusting the properties on the nodes instead.

## Editing the Save Interface

`res://addons/konado/template/ui_template/save_commponect/save_componect.tscn` is the save interface scene, and you can customise the save interface by modifying this file.
